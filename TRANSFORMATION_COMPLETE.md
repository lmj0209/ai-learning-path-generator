## 💬 TROUBLESHOOTING

### Q: Dashboard says "site can't be reached"
**A:** Server needs to start. Wait a moment or run:
```powershell
python run.py
```

### Q: I see old pink/yellow design
**A:** Hard refresh (Ctrl + Shift + R) or clear cache

### Q: Forms not submitting
**A:** Check browser console for errors (F12)

### Q: Text is hard to read
**A:** Adjust opacity in glassmorphic.css:
```css
--glass-bg: rgba(255, 255, 255, 0.08); /* Increase from 0.05 */
```

### Q: Glass effect not visible
**A:** Make sure your browser supports backdrop-filter:
- Chrome 76+
- Firefox 103+
- Safari 9+
- Edge 79+
